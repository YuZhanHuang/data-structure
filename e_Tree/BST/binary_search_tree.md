# Binary Search Tree Complexities

## Time Complexity

| Operation | Best Case Complexity | Average Case Complexity | Worst Case Complexity |
|-----------|----------------------|-------------------------|-----------------------|
| Search    | O(log(n))            | O(log(n))               | O(n)                  |
| Insertion | O(log(n))            | O(log(n))               | O(n)                  |
| Deletion  | O(log(n))            | O(log(n))               | O(n)                  |
